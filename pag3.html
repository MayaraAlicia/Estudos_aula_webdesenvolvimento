O que é CSS?

Capitulo 1

O CSS é uma linguagem de estilo usada para descrever a apresentação de um documento HTML. Ele permite que os desenvolvedores controlem a aparência de elementos HTML, como cores, fontes, espaçamento e layout. O CSS é uma parte fundamental do desenvolvimento web e é amplamente utilizado para criar sites visualmente atraentes e responsivos.

Capitulo 2

O CSS é composto por regras de estilo, que consistem em um seletor e um bloco de declarações. O seletor é usado para selecionar os elementos HTML aos quais as regras de estilo serão aplicadas, enquanto o bloco de declarações contém as propriedades e valores que definem a aparência dos elementos selecionados.

Capitulo 3

O CSS pode ser aplicado de várias maneiras, incluindo estilos embutidos, estilos internos e estilos externos. Os estilos embutidos são aplicados diretamente aos elementos HTML usando o atributo "style". Os estilos internos são definidos dentro da tag style no cabeçalho do documento HTML. Já os estilos externos são definidos em um arquivo CSS separado e vinculados ao documento HTML usando a tag link.
